### Анализ результатов A/B-теста в приложении ретейлера SkyLenta:
>#### Оценка эффективности новой механиники пуш-уведомлений и ее влияния на бизнес-метрики.
* **Задача:**
  * Проверка статистической значимости изменений конверсии и среднего чека после внедрения новой механиники пуш-уведомлений.
  * Построение какулятора прогнозной выручки.
* **Инструменты и методы:** Python (Pandas, Numpy, Matplotlib, Seaborn, Scipy), Excel, EDA, проверка гипотез, расчет p-value, оценка мощности теста.
* **Результат:**
  * Проведен комплексный анализ влияния пуш-уведомлений на ключевые бизнес-показатели (средний чек, конверсия).
  * Разработан интерактивный калькулятор в Excel для оценки результатов А/В-теста с возможностью детализации по конкретным городам и торговым точкам.
  * На основе полученных данных сформированы рекомендации по масштабированию теста.
  * Исследование выполнено в Jupyter Notebook, итоговый отчет представлен в формате Excel, полностью готовом для принятия управленческих решений.
