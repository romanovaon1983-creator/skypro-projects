### Исследование жизненного цикла клиентов и факторов удержания фитнес-сети SkySport
* **Бизнес-задача:** Провести комплексный анализ клиентской базы фитнес-сети для выяления закономерностей оттока. Оценить эффективность маркетинговой акции. Сегментировать аудиторию по уровню лояльности.
* **Навыки и инструменты:**
  * **Python (Pandas, Numpy):** глубокая предобработка данных, работа с пропусками и агрегациями.
  * **Matplotlib & Seaborn:** построение распределений для визуализиции когорт.
  * **Аналитические методы:** оформление анализа с логичными выводами в Jupyter Notebook .
* **Результат:** Сформирован аналитический отчёт с детальной сегментацией клиентов. Выявлены "точки роста" в маркетинговой стратегии и  определены сегменты с максимальным птенциалом удержания.
