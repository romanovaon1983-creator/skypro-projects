* **Задача:** оценить динамику пользовательской базы, активность игроков, финансовые результаты и показатели вирусного роста.
* **Иструменты:** PostgreSQL (CASE WHEN, агрегации, JOIN-ы, CTE/подзапросы), Excel (постобработка, визуализация).
* **Результат:** подготовлена презентация с основными выводами и визуализациями, демонстрирующая динамику пользовательской активности и влияние маркетинговых факторов на выручку игры.
